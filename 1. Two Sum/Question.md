## Example 1:

<code>Input: n=5, bad = 4</code>

Output: 4

Explanation:

-  call isBadVersion(3) -> false

-  call isBadVersion(5) -> true

-  call isBadVersion(4) -> true

Then 4 is the first bad version.


## Example 2:

<code>Input: n=1, bad = 1</code>

Output: 1

## Constraints:
* 1 <= bad ><=  n <= 231 - 1
